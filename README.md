# Rock Sites Explorer

A bilingual Django web application for showcasing and exploring rock and heritage sites in Saudi Arabia.
The platform allows users to browse locations, view image galleries, search for places, and access location details through Google Maps integration.

##  Live Website

https://waleedtw.pythonanywhere.com/ar/

##  Features

* Arabic & English support
* Search for places by name or region
* Detailed pages for each site
* Image galleries for locations
* Google Maps integration
* Responsive modern design
* Admin dashboard for managing places and images
* Custom ordering for displayed locations

##  Built With

* Python
* Django
* SQLite3
* HTML / CSS / JavaScript
* WhiteNoise
* PythonAnywhere (Deployment)

##  Project Structure

```bash
rock_sites/
│
├── config/          # Django project settings
├── places/          # Main application
├── templates/       # HTML templates
├── staticfiles/     # Static assets
├── media/           # Uploaded images
└── manage.py
```


##  Main Functionality

The application allows administrators to add and organize rock sites with:

* Arabic and English names
* Descriptions
* Regions
* Coordinates
* Main images
* Additional gallery images

Users can browse locations and open them directly in Google Maps.
